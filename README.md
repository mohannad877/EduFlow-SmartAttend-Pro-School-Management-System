## 🇬🇧 English Version

### 🚀 Overview
**EduFlow SmartAttend Pro** is an enterprise-grade school management solution built with Flutter. It focuses on automating complex administrative tasks such as intelligent timetable generation and student attendance tracking using modern technologies like QR/Barcode scanning and sophisticated scheduling algorithms.

### 🛠 Technical Architecture
The project follows **Clean Architecture** principles and the **BLoC (Business Logic Component)** pattern for state management, ensuring a highly maintainable and scalable codebase.
- **State Management:** Flutter BLoC & Riverpod
- **Database:** Drift (SQLite) for high-performance reactive persistence
- **Dependency Injection:** Provider/Riverpod
- **Localization:** i18n support (ARB) for Arabic and English

### ✨ Key Features
- 🧠 **Intelligent Scheduling:** Automated timetable generation with advanced conflict resolution algorithms.
- 📱 **Smart Attendance:** Attendance tracking via high-speed Barcode and QR code scanning.
- 📊 **Advanced Analytics:** Detailed reporting system with PDF and Excel export capabilities.
- 🔒 **Security & Audit:** Comprehensive audit logs and role-based access control (RBAC).
- ☁️ **Data Management:** Automated background backups and data recovery systems.
- 🌍 **Full Localization:** Native support for Arabic and English RTL/LTR layouts.

## 🇸🇦 النسخة العربية

### 🚀 نظرة عامة
**EduFlow SmartAttend Pro** هو حل متكامل لإدارة المدارس بمستوى احترافي، تم بناؤه باستخدام تقنية Flutter. يركز النظام على أتمتة المهام الإدارية المعقدة مثل توليد الجداول المدرسية الذكية وتتبع حضور الطلاب باستخدام تقنيات حديثة مثل مسح الباركود (Barcode/QR) وخوارزميات الجدولة المتقدمة.

### 🛠 الهندسة التقنية
يعتمد المشروع على مبادئ **Clean Architecture** ونمط **BLoC** لإدارة الحالة (State Management)، مما يضمن كوداً برمجياً قابلاً للصيانة والتوسع بشكل كبير.
- **إدارة الحالة:** Flutter BLoC & Riverpod
- **قواعد البيانات:** Drift (SQLite) للتخزين التفاعلي عالي الأداء.
- **حقن التبعيات:** Provider/Riverpod.
- **التدويل (Localization):** دعم كامل للغتين العربية والإنجليزية.

### ✨ المميزات الرئيسية
- 🧠 **الجدولة الذكية:** توليد تلقائي للجداول المدرسية مع خوارزميات متقدمة لحل التعارضات.
- 📱 **التحضير الذكي:** تتبع الحضور والغياب عبر مسح الباركود والـ QR بسرعة عالية.
- 📊 **تحليلات متقدمة:** نظام تقارير تفصيلي مع إمكانية التصدير بصيغ PDF و Excel.
- 🔒 **الأمان والرقابة:** سجلات مراجعة شاملة (Audit Logs) وإدارة صلاحيات المستخدمين.
- ☁️ **إدارة البيانات:** نظام نسخ احتياطي تلقائي في الخلفية واستعادة البيانات.
- 🌍 **دعم اللغات:** دعم كامل للغة العربية والإنجليزية مع مراعاة اتجاهات النص (RTL/LTR).
